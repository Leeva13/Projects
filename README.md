### **Emotionalee: AI-Powered Mental Health Platform**  
[![Java](https://img.shields.io/badge/Java-Spring%20Boot-brightgreen)](https://spring.io/)
[![React](https://img.shields.io/badge/Frontend-React.js-blue)](https://react.dev/)
[![Python](https://img.shields.io/badge/Python-NLP-yellow)](https://www.python.org/)  

**Description**:  
A full-stack web application designed to help users track their mental health with AI-driven insights. Features include real-time mood tracking, personalized recommendations, and NLP-based text analysis.  

**Tech Stack**:  
- **Backend**: Spring Boot (Java), REST APIs  
- **Frontend**: React.js, Bootstrap  
- **AI/ML**: Python (NLTK, TensorFlow) for sentiment analysis  
- **Database**: MySQL  

**Key Features**:  
✅ Real-time mood journal with AI feedback  
✅ Secure JWT authentication  
✅ Interactive dashboards for progress tracking  
✅ Integration with Telegram API for reminders
